سازگاری فروشگاه های ووکامرس با موتور جستجوی محصول تُرُب

== Description ==
با استفاده از این افزونه می توانید محصولات فروشگاه ووکامرسی خود را برای موتور جستجوی تُرُب آماده کنید .
این افزونه نیازی به هیچ تنظیمی ندارد و فقط کافیست آن را فعال کنید .
تنها در نظر داشته باشید که برای محصولات متغییر یا چند قیمتی باید یک حالت را به عنوان پیش فرض در نظر بگیرید که در منوی افزونه آموزش آن نمایش داده شده است  .
نکته مهم : امکانات این افزونه به صورت توکار و رایگان در [قالب فروشگاهی کانی شاپ](https://kanishop.ir) وجود دارند .

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woo-torob` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. No need for any setting
4. Enjoy :)
